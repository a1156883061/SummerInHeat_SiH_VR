# Release Notes

Release notes are stored under `release-notes/versions/`.

- Use `versions/next.md` for unreleased changes.
- When publishing a release, rename `next.md` to `vX.Y.Z.md`.
- Keep each version file bilingual, with English first and Chinese second.

# 更新说明

更新说明统一存放在 `release-notes/versions/` 目录。

- 未发布改动写入 `versions/next.md`。
- 正式发布时，将 `next.md` 重命名为 `vX.Y.Z.md`。
- 每个版本文件保持中英双语，英文在前，中文在后。
