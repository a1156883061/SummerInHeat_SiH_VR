# Summer in Heat VR Mod（中文说明）

> [!IMPORTANT]
> 本项目是 **UnityVRMod** 框架的特化版本，专为 Miconisomi 的 **Summer in Heat**（夏のサカり）定制。

---

## ✨ 功能特性

*   **原生 6DOF VR 体验**：基于 **OpenXR** 的完整六自由度追踪。
*   **沉浸式交互**：专为 VR 手柄设计的 3D UI 交互系统。
*   **高级移动系统**：VR 传送和舒适视角控制（快照/平滑转向）。
*   **动态 UI 管理**：可在 VR 中重新定位和缩放游戏内 UI 面板。
*   **无缝视角切换**：优化的第一人称相机绑定和场景过渡。

---

## 🛠️ 环境要求

*   **游戏**：Summer in Heat（夏のサカり）
*   **框架**：[BepInEx 6 x64](https://github.com/BepInEx/BepInEx/releases) pre.2
*   **测试设备**：Meta Quest 3（通过 Meta Quest Link / Air Link）
    *   *注：其他头显可能通过 OpenXR 兼容，但未经过正式测试。*

---

## 🚀 安装

1. **安装 BepInEx 6**：确保游戏目录中已安装 x64 版本的 BepInEx 6。
2. **部署模组文件**：将发布包中的内容解压到 `BepInEx/` 文件夹。
3. **配置**：
   *   首次启动游戏以生成初始配置文件。
   *   （可选）将发布包 `config/` 中的 `.cfg` 文件复制到 `GameData/BepInEx/config/`。
4. **启动 VR**：
   *   启动 VR 运行时（如 Meta Quest Link 或 Virtual Desktop）。
   *   启动游戏，按 **F11** 切换 VR 模式（可在配置文件中自定义按键）。

## 🎮 操作说明 (Oculus/Meta Touch)

> [!TIP]
> `OpenXR Control Hand` 控制移动和交互手。`OpenXR UI Panel Follow Hand` 控制 UI 面板跟随哪只手。双击 **X(左)/A(右)** 可在游戏中动态切换面板跟随手。

| 操作 | 按键 |
| :--- | :--- |
| **传送瞄准** | **右摇杆 ↑**（始终右手） |
| **确认传送** | **右手扳机**（始终右手） |
| **快照转向** | **右摇杆 ←→**（始终右手） |
| **平滑转向** | **按住右摇杆** + ←→ |
| **平滑移动/行走** | **左摇杆**（头显方向，速度可配置） |
| **拖拽视角** | **按住握把（任意手）** + 拖拽 |
| **切换面板锚定/跟随** | **单击 Y(左) / B(右)** |
| **切换面板跟随手** | **双击 X(左) / A(右)** |
| **切换面板显隐** | **单击 X(左) / A(右)** |
| **UI 点击** | **任意手扳机**（按下的那只手自动切换） |
| **面板缩放/拖拽** | **扳机（边界处）** + 拖拽 |
| **触摸交互（Petting）** | **扳机（任意手）** + 手柄靠近目标 |
| **子相机模式** | **长按 A（右手）** |
| **移动子相机** | **右摇杆**（子相机模式下） |
| **旋转子相机** | **按住右摇杆** + **右摇杆** |
| **子相机高度** | **点按 A（升）/ B（降）**（子相机模式下） |

### UI 交互

- **任意手扳机都能点击**：左手或右手扳机均可点击 UI——按下瞬间锁定到该手，直到另一只手点击才会切换。
- **面板跟随手**：双击 X(左) 或 A(右) 切换 UI 面板跟随的手。面板跟随手独立于控制手（移动/传送用的手）。
- **面板显隐**：单击 X(左) 或 A(右) 切换 UI 面板的显示/隐藏。Hybrid2D 场景中此功能禁用。

### 拖拽视角

1. 按住**任意手柄的握把键**。
2. 移动手柄拖动世界。
3. 松手停止。

### 触摸交互 (Petting)

1. 将任意手柄靠近目标身体部位。
2. 检测到有效目标时，图标变为**蓝色**。
3. 扣下**扳机**触发交互——图标变为**橙色**。
4. 仅在自由模式生效。

### 子相机控制

1. **长按 A** 切换子相机移动模式 开/关。
2. 移动模式下，移动**右摇杆**控制子相机位置。
3. 按住**右摇杆按下** + 移动摇杆旋转子相机。
4. 移动模式下**点按 A** 升高、**点按 B** 降低子相机高度。
5. 移动模式开启时显示红色边框。

## ⚠️ 当前状态与路线图（2026年2月）

> [!NOTE]
> 此模组正在积极开发中。以下是目前已知的限制和计划更新：

*   **硬件兼容性**：已针对 Quest 3 优化，其他头显正在评估中。
*   **ADV 场景**：目前以 2D 投影方式显示，VR 中视觉效果可能有所不同。
*   **UI 集成**：某些对话框图标需通过主 UI 面板交互，无法直接点击。
*   **性能注意**：高画质设置和特效会显著影响 VR 帧率，请谨慎调整以确保稳定性。
